## Purpose

This document records the conjectured correspondence between Candidate C and Candidate D as a disciplined organizing principle.

The correspondence is useful only if it prevents isolated prose development. It does not supply proof content.

## Candidate surfaces

Candidate C:

```text
Einstein-AdS / fluid-gravity-style bulk
  -> boundary/envelope hydrodynamic projection
  -> recovered fluid equation, known deformation, or failure
```

Candidate D:

```text
Heller-Einstein interface / projection bulk or latent state
  -> typed observer trace
  -> recovered fluid equation, known deformation, kernel-valued dynamics, not-configured, or failure
```

## Conjecture statement

There may exist a correspondence between the Candidate C boundary/envelope projection and the Candidate D typed observer trace such that both recover the same effective fluid equation on the boundary/envelope under a shared declared limit.

This is a C0 conjectural organizing principle. It is not a theorem target in v0.2.

## Morphism class is intentionally open

v0.2 does not commit to whether the correspondence is:

- bijective;
- surjective;
- injective;
- functorial after quotienting;
- approximate under a limit;
- comparison-only;
- absent.

The morphism class must be determined by structure, not imposed by language.

## Evidence that would support the correspondence

Evidence in favor would include:

- both candidates recover the same target equation under explicitly matched limits;
- both candidates recover the same known deformation with matched parameters;
- Candidate C stress/boundary variables can be mapped to Candidate D trace variables without changing claim grade;
- Candidate D observer variables reproduce Candidate C hydrodynamic observables in a declared limit;
- perturbations or null cases produce compatible failure/success behavior.

## Evidence against the correspondence

Evidence against would include:

- dimension mismatch that cannot be repaired;
- Candidate C recovers a deterministic PDE while Candidate D only recovers a kernel-valued observer process;
- Candidate C recovers a relativistic conformal fluid while Candidate D recovers a non-fluid trace structure;
- one candidate recovers Navier-Stokes or a known deformation while the other fails under matched assumptions;
- required maps are not definable at the pinned upstream versions.
